# TEK1314-2025-Kel-F2-Kelas-F2
Project ini adalah implementasi keamanan jaringan untuk simulasi dan analisis serangan berbasis DNS dalam skala kecil. Sistem ini menggunakan beberapa virtual machine yang terdiri dari Gateway Router, DNS Server, Security Onion sebagai Intrusion Detection System (IDS), dan mesin attacker menggunakan Cybersecurity LabVM Workstation. Lingkungan jaringan menggunakan subnet 192.168.70.0/24, dan lingkungan ini dibuat untuk melakukan pengujian serangan DNS secara terkontrol serta mengamati bagaimana aktivitas jaringan dapat dideteksi dan dianalisis sebagai bagian dari penerapan konsep defensive security.
